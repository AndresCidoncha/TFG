Preferences

Opens the Preferences Dialog. Preferences are your software operation settings that are stored in a file named ActivMedia.pref, which is located either in your C:\Windows directory or in your Linux ~/.ActivMedia directory. If you remove that file, ActivMedia software automatically reloads it using default settings, which can sometimes be a fix if your ActivMedia software gets broken. Edit the ActivMedia.pref text file at your own risk.

Sounds Directory

The Sounds Directory is the path to where you normally store your original sound files. Click in the adjacent text box and type the directory name or click the ellipsis (...) button and use the dialog to select the Sounds Directory pathname. This is the directory of sounds that is loaded into the Sound Files window when you start AmigoSounds.

AMS Directory

This is the default path to the directory where you normally store your ActivMedia Sound files (AMS) for AmigoBot. Click in the adjacent text box and type the directory name or click the ellipsis (...) button and use the dialog to select the AMS directory pathname.

Robot Connection Port

Click and hold, draw down, and release to select from the pull down menu the name of your PC's serial port through which you will download ActivMedia Sound files to your AmigoBot. Available Window9x/NT(R) port names are COM1 through COM4; Linux users get /dev/ttyS0 through /dev/ttyS3. Select None to force the Connection dialog to appear each time you want to connect AmigoSounds with AmigoBot.

The Robot Connection Port names one of several possible serial ports on your PC to which you have connected the AmigoLeash. For all sound downloads, the other end of the AmigoLeash only goes into the System Serial Port underneath AmigoBot next to the Main Power switch and Charge Port.

Show Sound Lengths

Click the button to enable (red) or disable (clear; default) the feature that calculates how much time, in seconds, each sound in the Sound Files window will impact AmigoBot's Available Time, if transferred to the AMS window for eventual download to your robot. Calculating sound lengths takes a lot of time in itself, so you may want to disable this option.

OK, Save, Defaults, and Cancel

Click to select one of these dialog options: OK saves your changes in the ActivMedia.pref file and closes the dialog; Save puts your changes into the ActivMedia.pref file, but does not close the dialog; Default restores each of the options to their original state when you first installed the software; Cancel closes the dialog without saving your changes.
