AmigoSounds File Menu

The AmigoSounds File menu is where you load and save ActivMedia Sound files (AMS) and choose the directory from which you may add and exchange sounds into the AMS Sounds window.

New

Clears the current AMS window in preparation for an entirely new ActivMedia Sound file (AMS). You'll be asked to Save your work if you had made any changes to the current AMS. If so, choose Save in the Save File dialog to overwrite the original file (default) or type in a new pathname to create a new AMS (the ".ams" filename suffix is added automatically to the filename if you don't include it). Choose Cancel in the dialog to retain the original AMS on disk, ignoring any edits you may have made to it in the Sounds window, and enabling a fresh (clear) Sounds window.

Open

Same as the Load AMS button on the AmigoSounds console, Open lets you select and load an ActivMedia Sound file into the AMS window for editing and eventual download to AmigoBot.

You'll be asked to Save your previous work if you had made any changes to the current AMS. If so, choose Save in the dialog to overwrite the original file (default); if it's a New AMS, you'll be asked to type in a pathname to create a new ".ams" sound file. Choose Cancel in the dialog to retain the original file on disk, ignoring any edits you may have made to it in the AMS window.

Open starts a file selection dialog from which you select an AMS from the default ActivMedia Sounds directory or elsewhere. Choose Okay to load the selected file into the AMS window, or Cancel to ignore the selection and retain the current sound file.

Close

Same as New.

Save

Saves the sound file you've assembled and edited in the AMS window to a permanent file in your chosen AMS directory on disk. The Save As... dialog will appear asking you for a directory path and filename if it's a New AMS. Otherwise, this operation automatically overwrites the Opened AMS with the sounds as it appears in the AMS window.

Save As...

Enables a dialog in which you specify a name and target directory for saving the AMS that you have assembled and edited in the AMS window. Note that the filename suffix ".ams" automatically gets appended to the given filename, if you don't add it to the name yourself.

Download to AmigoBot

Downloads the current AMS to your AmigoBot. See a description of the Download console button for details.

Choose Sounds Directory

Opens a dialog from which you select the directory on one of your disks where you store the sound recordings (".wav", ".au", and other sound files) for addition to the current AMS in the Sounds window. The various sound filenames appear in AmigoSounds' Sound Files window.

If you have enabled Show Sound Lengths in your Preferences, AmigoSounds will take several moments to calculate and display the length of time in seconds for each sound in the adjacent Length window. Please be patient.

CAUTION:

Occasionally, the Microsoft Windows(R) version of the accessory program SOX, will cause an error by performing an "illegal operation" when calculating length and time for some sounds. Please Close the error notice. AmigoSounds also will notify you that an error happened, but it's usually safe to continue. We apologize for the inconvenience, but such errors currently are unavoidable.

Quit

Same as the Close Box in the upper right corner of the main window. You'll be asked to Save your work, if you had made any changes to the current AMS window. If so, choose Save in the dialog to overwrite the original file (default) or, if it's a New AMS, you'll be asked to type in a new pathname to create a new ".ams" sound file. Choose Cancel in the dialog to retain the original file on disk, ignoring any edits you may have made in the AMS window.

You'll also be asked if you want to "Really quit?" Choose Yes to shut down AmigoSounds and closes all accessory files and programs. No resumes the program where you left off.
